# Historian Comparison

Use these author profiles to keep comparisons sharp and compact.

- Ian Kershaw: strongest default anchor for political trajectory, structure, charisma, and the leader myth
- Peter Longerich: detailed biography with strong value for chronology, power structure, and decision-making
- Volker Ullrich: useful for political personality, style, public self-presentation, and narrative readability
- John Toland: expansive classic biography; useful as a broad comparative source, but not always the first evidentiary anchor
- Joachim Fest: historically important and still useful, but treat it as part of the comparison set rather than the sole authority
- Ron Rosenbaum: interpretive debate and historiography, not the first source for precise factual anchoring

Comparison pattern:

1. Name the agreement.
2. Name the difference in emphasis.
3. Cite the stronger anchor first.

Example:

- Chinese: `Kershaw 更强调结构性危机与领袖神话，Ullrich 则更突出政治人格与公众表演。`
- English: `Kershaw puts more weight on structural crisis and the leader myth, while Ullrich gives more attention to political personality and public performance.`
