# Quote Handling

Use this sequence whenever a user asks for a quote or quote verification:

1. Decide whether the claim is a direct quote, paraphrase, historian summary, memoir recollection, or unknown internet attribution.
2. If provenance is unclear, say so immediately.
3. Keep any quoted text short.
4. Mention the source type and context.

Preferred labels:

- `direct quote`
- `paraphrase`
- `historian summary`
- `postwar recollection`
- `unverified attribution`

What to avoid:

- treating a widely repeated internet line as authentic without a source
- blending wording from multiple translations into one "original" quote
- giving a quote in first-person performance mode
